# Bengkel Sitepu - Solusi Perawatan Mobil Terpercaya

Bengkel Sitepu adalah aplikasi web modern untuk bengkel mobil profesional yang dibangun menggunakan Next.js. Website ini dirancang untuk memberikan pengalaman pengguna yang premium, estetik, dan informatif bagi pelanggan yang mencari layanan perawatan kendaraan berkualitas tinggi.

## Fitur Utama

- **Premium UI/UX**: Desain modern "Midnight & Amber" dengan animasi halus menggunakan Framer Motion.
- **Giant SEO**: Optimasi mesin pencaraian mendalam dengan JSON-LD Structured Data, sitemaps, dan metadata dinamis.
- **Layanan Terintegrasi**: Informasi lengkap mengenai Ganti Oli, Tune Up, Servis AC, Rem, dan Overhaul.
- **Booking Konten**: Form reservasi servis yang intuitif dan integrasi WhatsApp untuk komunikasi instan.
- **Responsive Design**: Tampilan yang sempurna di semua perangkat (Desktop, Tablet, Mobile).

## Teknologi yang Digunakan

- [Next.js](https://nextjs.org/) (App Router)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/)
- [TypeScript](https://www.typescriptlang.org/)

## Cara Menjalankan Secara Lokal

1. Clone repositori:
   ```bash
   git clone git@github.com:MarioSitepu/car-repair-shop.git
   ```
2. Instal dependensi:
   ```bash
   npm install
   ```
3. Jalankan server pengembangan:
   ```bash
   npm run dev
   ```
4. Buka [http://localhost:3000](http://localhost:3000) di browser Anda.
